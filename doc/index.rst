Mesure de température par LIF pour la convection thermique
==========================================================

Bienvenue dans la documentation de l'équipe PlumeX_ pour les acquisitions et traitements
de fluorescence induite par laser (LIF).
Cette documentation explique le protocole que l'on utilise pour mener une expérience de LIF,
ainsi que l'utilisation des scripts Matlab écrits par Clément Toupoint pendant son
`séjour post-doctoral <https://plumexlyon.github.io/arrivee-de-clement-toupoint.html>`_.

Les scripts Matlab, ainsi que le code source de cette documentation, sont disponibles dans
le dépôt `lif_matlab <https://github.com/plumexlyon/lif_matlab>`_ sur GitHub.
